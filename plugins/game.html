Game¶

Patient and impatient pedestrians in a spatial game.

Patient and impatient pedestrians in a spatial game \((S, f)\) for egress congestion between players \(P \subset A\) with set of strategies \(S\) and payoff function \(f : S \times S \mapsto \mathbb{R}\).

[Heliovaara2013], [VonSchantz2014]

Set of strategies

\[\begin{split}S &= \{ \text{Impatient}, \text{Patient} \} \\ &= \{ 0, 1 \}\end{split}\]

Payoff matrix / function

\[\begin{split}f(s_i, s_j) = \left[\begin{matrix}\left ( \frac{T_{aset}}{T_{ij}}, \quad \frac{T_{aset}}{T_{ij}}\right ) & \left ( -1, \quad 1\right )\\\left ( 1, \quad -1\right ) & \left ( 0, \quad 0\right )\end{matrix}\right]\end{split}\]

Estimated evacuation time for an agent

\[T_i = \frac{\lambda_i}{\beta},\]

where

  • \(\beta\) is the capacity of the exit door
  • \(\lambda_i\) number of other agents closer to the exit.

Average evacuation time

\[T_{ij} = \frac{(T_i + T_j)}{2}.\]

Effect on agents

  • \(k\)
  • \(\tau_{adj}\)
  • \(\sigma_{force}\)

crowddynamics.plugins.game.game.best_response_strategy[source]¶

Best Response Strategy

New strategy is selected using best response dynamics which finds the strategy that minimizes loss using the payoff function

\[s_{i} = \underset{s \in S}{\arg \min} \sum_{j \in N_{i}^{neigh}} f(s, s_{j})\]

where

  • \(N_{i}^{neigh} \subset P \setminus \{P_{i}\}\) set is eight closes agents excluding the agent itself at maximum skin-to-skin distance of \(0.40 \ \mathrm{m}\) from agent \(i\).
