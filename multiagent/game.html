5.10. Game Theoretical Models¶

5.10.1. Spatial Game for Egress Congestion¶

Spatial game for egress congestion between players \(P \subset A\)

\[(S, f)\]

where

  • \(S\) is the set of strategies
  • \(f : S \times S \mapsto \mathbb{R}\) is the payoff function

Individual strategy is denoted

\[s_{i} \in S, \quad i \in P\]

5.10.2. Payoff Matrix¶

  Impatient Patient
Impatient \(T_{aset}/T_{ij}, T_{aset}/T_{ij}\) \(-1, 1\)
Patient \(1, -1\) \(0, 0\)

Set of strategies

\[\begin{split}S &= \{ \text{Impatient}, \text{Patient} \} \\ &= \{ 0, 1 \}\end{split}\]

Payoff function

\[\begin{split}f(s_i, s_j) = \begin{cases} \begin{cases} T_{aset}/T_{ij} \\ 1 \end{cases} \\ \begin{cases} -1 \\ 0 \end{cases} \end{cases}\end{split}\]

Available Safe Egress Time

\[T_{aset}(t) = T_{0} - t,\]

where

  • \(T_{0}\) is initial available safe egress time
  • \(t\) is current simulation time.

Estimated evacuation time for an agent

\[T_i = \frac{\lambda_i}{\beta},\]

where

  • \(\beta\) is the capacity of the exit door
  • \(\lambda_i\) number of other agents closer to the exit.

Average evacuation time

\[T_{ij} = \frac{(T_i + T_j)}{2}.\]

5.10.3. Strategy Selection¶

New strategy is selected using best response dynamics

\[s_{i} = \underset{s \in S}{\arg \min} \sum_{j \in N_{i}^{neigh}} f(s, s_{j})\]

where

  • \(N_{i}^{neigh} \subset P \setminus \{P_{i}\}\) set is eight closes agents at maximum skin-to-skin distance of \(0.40 \ \mathrm{m}\) from agent \(i\)
  • \(\arg \min\) function argmin

5.10.5. Agents Closer to Exit¶

Function mapping players to number of agents that are closer to the exit is denoted

\[\lambda : P \mapsto [0, | P | - 1].\]

Ordering is defined as the distances between the exit and an agent

\[d(\mathcal{E}_i, \mathbf{x}_{i})\]

where

  • \(\mathcal{E}_i\) is the exit the agent is trying to reach
  • \(\mathbf{x}_{i}\) is the center of the mass of an agent

For narrow bottlenecks we can approximate the distance

\[d(\mathcal{E}_i, \mathbf{x}_{i}) \approx \| \mathbf{c} - \mathbf{x}_{i} \|\]

where

  • \(\| \cdot \|\) is euclidean metric
  • \(\mathbf{c}\) is the center of the exit.

Algorithm

  1. Sort by distances to map number of closer agents to player

    \[\boldsymbol{\lambda}^{-1} = \underset{i \in P}{\operatorname{arg\,sort}} \left( d(\mathcal{E}_i, \mathbf{x}_{i}) \right)\]

  2. Sort by players to map player to number of closer agents

\[\boldsymbol{\lambda} = \operatorname{arg\,sort} (\boldsymbol{\lambda}^{-1})\]
