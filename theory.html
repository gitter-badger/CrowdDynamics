Density¶

Crowd density \(\rho\) is the number of agent per unit of area \(\mathrm{P / m^{2}}\). Realistic values range from \(0\) to \(10\) people per square metre. Density ranges can be classified in increasing order [gkstill]

Class min max
Free flow \(0\) \(1\)
Stable \(1\) \(2.5\)
Capacity \(2.5\) \(3.5\)
Unstable \(3.5\) \(5.0\)
Congested \(\geq 5.0\)  

Another way to define density in unit-less quantity is to define it to be the percentage of area occupied. This has value range \(0\) to \(1\).

Flow¶

Crowd flow which has similarities with granular flow, gas kinetics and fluid-dynamics measures the magnitude and how the crowd flows. Rate of flow is defined as number of agents per second

\[J = \frac{\Delta N}{\Delta t}\]

There are several types of crowd flow, which can be classified [Duives2014]

Mathematically the direction of the flow can be derived from the dot product. For agents with velocities \(\mathbf{v}_{i}\) and \(\mathbf{v}_{j}\) the dot product

\[\begin{split}\mathbf{v}_{i} \cdot \mathbf{v}_{j} = \|\mathbf{v}_{i}\| \|\mathbf{v}_{j}\| \cos(\varphi) \\\end{split}\]

where \(\varphi\) is the angle between the agents. In order to determine the type of the flow we must calculate the value of angle \(\varphi\) between neighbouring agents.

Pressure¶

Pressure \(p\) in crowds created by contact forces \(\mathbf{f}_{c}\) from agents pressing onto each others.

\[p \propto \sum_{i \in N_{neighbor}} \mathbf{f}_{c_i}\]

Fundamental Diagram¶

Fundamental diagram offers an empirical relationship between density \(\rho\) and velocity \(\mathbf{v}\) of pedestrian movement. It can also be defined as the relationship between density \(\rho\) and flow \(\Phi = \rho \mathbf{v}\). [Seyfried2005]
