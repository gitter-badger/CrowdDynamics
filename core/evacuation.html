Evacuation related functions

crowddynamics.core.evacuation.evacuation.agent_closer_to_exit[source]¶

Amount of positions (agents) closer to center of the exit.

  1. Denote \(i\) as indices of the positions \(\mathbf{x}\).
  2. Distance from narrow exit can be estimated
\[d_i = \| \mathbf{c} - \mathbf{x}_{i} \|\]
  1. By sorting the values \(d_i\) by its indices we obtain array
\[a_i = \underset{i \in P}{\operatorname{arg\,sort}}(d_i)\]

where

  • Values: Indices of the positions sorted by the distance from the exit
  • Indices: Number of positions closer to the exit
  1. By sorting the values \(a_i\) by it indices we obtain array
\[\lambda_i = \operatorname{arg\,sort} (a_i)\]

where

  • Values: Number of positions closer to the exit
  • Indices: Indices of the positions
Parameters:
  • c_door (numpy.ndarray) – Center of the exit \(\mathbf{c}\).
  • position (numpy.ndarray) – Positions \(\mathbf{x}\) of the agents.
Returns:

Array \(\lambda_i\)
Return type:

numpy.ndarray
crowddynamics.core.evacuation.evacuation.narrow_exit_capacity[source]¶

Estimation of the capacity of narrow exit.

Capacity estimation \(\beta\) of unidirectional flow through narrow bottleneck. Capacity of the bottleneck increases in stepwise manner.

Simple estimation

\[\beta_{simple} = c \left \lfloor \frac{d_{door}}{d_{agent}} \right \rfloor\]

More sophisticated estimation [Hoogendoorn2005a], [Seyfried2007a] when \(d_{door} \geq d_{agent}\)

\[\beta_{hoogen} = c \left \lfloor \frac{d_{door} - (d_{agent} - d_{layer})}{d_{layer}} \right \rfloor\]

We have relationship between simple and hoogendoorn’s estimation

\[\beta_{hoogen} = \beta_{simple}, \quad d_{layer} = d_{agent}\]
Parameters:
  • d_door (float) – Width of the door \(0\,\mathrm{m} < d_{door} < 3\,\mathrm{m}\).
  • d_agent (float) – Width of the agent \(d_{agent} > 0\,\mathrm{m}\).
  • d_layer (float, optional) – Width of layer \(d_{layer}\). Reference value: \(0.45\,\mathrm{m}\)
  • coeff (float) – Scaling coefficient \(c > 0\) for the estimation.
Returns:

Depending on the value of d_layer either simple or hoogen estimation is returned

  • None: Uses simple estimation \(\beta_{simple}\)
  • float: Uses Hoogendoorn’s estimation \(\beta_{hoogen}\)
Return type:

float
